# Parsing

		- `data/1524.txt` An example script
		- ... other files to come
	- `output/` contains output files
		- `output/1524poor.xml` Poorly xml script created by parse.php
		- ... other files to come
	- `parse.php` can be run with php parse.php data/1524.txt > output/1524poor.xml`
